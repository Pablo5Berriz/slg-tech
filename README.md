# slg-tech
SLG Tech est une boutique en ligne spécialisée dans la vente d’appareils informatiques et technologiques. Le projet permet de créer une plateforme e-commerce moderne pour vendre des ordinateurs, accessoires, composants, équipements réseau, périphériques et produits électroniques.

La plateforme est conçue pour offrir une expérience d’achat simple, rapide et adaptée aux particuliers comme aux petites entreprises. Elle inclut un catalogue produit, un panier, une gestion de commandes, un espace client et une interface administrateur.

SLG Tech peut évoluer vers une boutique complète avec gestion des stocks, paiements en ligne, promotions, livraison et service après-vente.

Fonctionnalités principales :

* Catalogue de produits technologiques
* Fiches produits détaillées
* Catégories et filtres
* Panier d’achat
* Paiement en ligne
* Gestion des commandes
* Gestion des stocks
* Espace client
* Interface administrateur
* Promotions et produits vedettes
